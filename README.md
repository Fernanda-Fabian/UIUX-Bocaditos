# Bocaditos – Repositorio de Diseño UI/UX

Este repositorio guarda los diseños del proyecto Bocaditos, como los wireframes, prototipos y estilos visuales usados en la interfaz.

## Estructura
- /wireframes: bocetos de baja fidelidad
- /prototipo-mediana: prototipo visual con navegación
- /justificacion: documento de diseño visual y tipográfico
- /capturas-versiones: evolución visual entre versiones
